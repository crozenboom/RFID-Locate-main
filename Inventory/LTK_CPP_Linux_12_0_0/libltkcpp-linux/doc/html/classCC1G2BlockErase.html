Detailed Description

Class Definition CC1G2BlockErase for LLRP parameter C1G2BlockErase.

Document Reference LLRP Specification Section 15.2.1.3.2.5

Document Reference LLRP Specification Section 16.3.1.3.2.5 

<p>MB is the memory bank to use. WordPtr is the starting word address. Word Count is the number of 16-bit words to be read. Access Password is the password used by the Reader to transition the tag to the secure state so that it can erase protected tag memory regions.</p> 


<p>Readers that do not support C1G2BlockErase

SHALL set CanSupportBlockErase to false in C1G2LLRPCapabilities. If such a Reader receives an ADD_ACCESSSPEC with an AccessSpec that contained this OpSpec parameter, the Reader SHALL return an error for that message and not add the AccessSpec.
