Detailed Description

Class Definition CImpinjXPCWords for LLRP parameter ImpinjXPCWords. 

<p>This custom parameter reports any Extended Protocol Control words that are backscattered during normal inventory.</p> 


<p>XPCWords: A word vector containing the XPC data. The order of delivery is XPC_W1, then XPC_W2.</p>
