Class Definition CReportBufferLevelWarningEvent for LLRP parameter ReportBufferLevelWarningEvent.

Document Reference LLRP Specification Section 13.2.6.5

Document Reference LLRP Specification Section 16.2.7.6.4 

<p>A Reader can warn the Client that the Reader's report buffer is filling up.  A Client can act upon this warning by requesting report data from the Reader, thereby freeing the Reader's report memory resources.</p> 


<p>A Reader

MAY send a report buffer level warning event whenever the Reader senses that its report memory resources are running short. The buffer level at which a warning is reported is Reader implementation dependent. A Client MAY act upon a report buffer level warning event by requesting report data from the Reader and thereby free report memory resources in the Reader.
