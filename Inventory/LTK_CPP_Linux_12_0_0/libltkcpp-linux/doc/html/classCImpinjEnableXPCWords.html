Detailed Description

Class Definition CImpinjEnableXPCWords for LLRP parameter ImpinjEnableXPCWords. 

<p>This custom parameter configures the reporting of Extended Protocol Control words.</p> 


<p>XPCWordsMode: Disabled by default. When set to enabled, any backscattered Extended Protocol Control words will be provided along with the TagReportData.</p>
