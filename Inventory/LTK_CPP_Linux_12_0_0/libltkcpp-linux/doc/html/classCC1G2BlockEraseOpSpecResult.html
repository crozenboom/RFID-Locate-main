Class Definition CC1G2BlockEraseOpSpecResult for LLRP parameter C1G2BlockEraseOpSpecResult.

Document Reference LLRP Specification Section 15.2.1.5.5.5

Document Reference LLRP Specification Section 16.3.1.5.5.5 

<p>Contains the result of a block erase operation.</p> 


<p>Readers that do not support C1G2 Block Erase

SHALL set CanSupportBlockErase to false in C1G2LLRPCapabilities. If such a Reader receives an ADD_ACCESSSPEC with an AccessSpec that contains this OpSpec parameter, the Reader SHALL return an error for that message and not add the AccessSpec.
