Detailed Description

Class Definition CTagReportContentSelector for LLRP parameter TagReportContentSelector.

Document Reference LLRP Specification Section 13.2.1.1

Document Reference LLRP Specification Section 16.2.7.1.1 

<p>This parameter is used to configure the contents that are of interest in TagReportData. If enabled, the field is reported along with the tag data in the TagReportData.</p>
