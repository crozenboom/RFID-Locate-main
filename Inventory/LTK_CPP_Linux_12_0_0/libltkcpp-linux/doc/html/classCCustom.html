Class Definition CCustom for LLRP parameter Custom.

Document Reference LLRP Specification Section 8.2

Document Reference LLRP Specification Section 16.2.9 

<p>Certain Messages and Parameter Sets within LLRP allow for the insertion of vendor defined parameters</p> 


<p>Clients

SHALL accept messages (except for CUSTOM_MESSAGE) that contain custom parameters but MAY ignore all custom parameters within these messages. Readers SHALL accept messages (except for CUSTOM_MESSAGE) that contain custom parameters and SHALL return an error when such parameters are unsupported.
